Projects

Hausa Sentiment Classifier

Developed a sentiment classification system for Hausa movie reviews using BiLSTM and FastText embeddings.

Societal Impact: Enables analysis of regional media trends and helps local organizations gauge public opinion.

Low-Resource Translation Model

Created a neural translation model for Hausa-English using transfer learning on Masakhane datasets.

Societal Impact: Promotes digital inclusion and improves access to knowledge for underrepresented language speakers.

Bias Detection in African Language Models

Investigated gender and regional bias in African NLP models, contributing to ethical AI frameworks.

Societal Impact: Supports fairer AI systems and responsible language model deployment in African contexts.
