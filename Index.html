About Me

Projects & Societal Impact

Hausa Sentiment Classifier

Role: Lead • Tools: Hugging Face, FastText, PyTorch

Built and evaluated a sentiment classifier for Hausa movie and product reviews, addressing class imbalance using targeted augmentation and careful annotation guidelines.

Societal Impact: Enables better sentiment insights for local businesses and researchers, helping content moderation and customer feedback analysis in Hausa-speaking communities.
GitHub

Aspect-Based Sentiment Analysis (ABSA)

Role: Model Developer • Tools: BiLSTM, Attention, FastText

Designed an ABSA pipeline to extract sentiment toward specific product or service aspects within reviews, combining neural attention mechanisms with rule-based heuristics.

Societal Impact: Helps small businesses and NGOs understand specific pain points in services (e.g., delivery, pricing, quality) and target improvements in local markets.
Paper / Repo

Low-Resource Text Cleaning & Toolkit

Role: Data Engineer • Tools: Python, Pandas

Created an open-source toolkit for cleaning and tokenizing texts in African languages, including normalization rules, stopword lists, and script handling.

Societal Impact: Lowers the barrier for researchers and practitioners to build language tools for underrepresented languages, fostering local research and technology access.
Toolkit

Reproducible NLP Pipelines with Docker

Role: Pipeline Engineer • Tools: Docker, GitHub Actions

Packaged experiments and training scripts into Docker containers and integrated CI to ensure reproducible results across collaborators and cloud/edge setups.

Societal Impact: Promotes transparency and reproducibility in research, which increases trust in model claims and helps collaborators in low-infrastructure settings reproduce experiments reliably.
Repo
